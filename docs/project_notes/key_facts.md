## Image Styles
- **Bio illustrations**: Watercolor style, warm sepia tones, narrative moments
- **Location panels**: Hand-watercolored engraving style (linework + subtle color washes)
  - Reference: https://mirka-h.blogspot.com/2015/02/imprinting-american-west-engravings.html
  - Techniques: parallel lines, cross-hatching, stippling; watercolor applied to complement not overwhelm linework
